# 사전

이 문서는 프로젝트 내에서 기여자들이 번역한 용어를 일관적으로 관리하기 위해 작성되었습니다. 여러 기여자가 동일한 용어를 다르게 번역하는 것을 예방하기 위해, 각 기여자는 번역 과정에서 사전을 참고하고, 번역을 마친 뒤에 사전에 대해서도 기여할 것을 권장드립니다.
